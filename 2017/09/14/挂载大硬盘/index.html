分区时，小硬盘和大硬盘的分区方法有所区别

大硬盘分区

大数据盘的分区和文件系统格式化和小盘都存在差异。大盘必须采用 GPT 分区格式， 不能再采用小盘使用的 MBR 分区格式。

  • MBR 分区格式：最大支持 2 TB 的磁盘。
  • GPT 分区格式：最大支持 18 EB。

大盘支持的文件系统如下:

操作系统 分区工具 文件系统
Linux parted ext4/xfs
Windows Windows磁盘管理 NTFS
